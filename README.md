 ExamHub
Project Overview
ExamHub is an Android application aimed at assisting Grade 12 students in Ethiopia with their preparations for national examinations. This interactive quiz-based platform allows students to choose subjects, answer multiple-choice questions, and receive immediate feedback on their performance, promoting effective study habits.

Problem Statement
Grade 12 students frequently depend on traditional paper-based study methods, which restrict self-assessment and immediate feedback. ExamHub tackles these issues by offering a modern, mobile-based solution for convenient and interactive exam preparation.

Core Features
Register Screen: Enables new users to create accounts for personalized access.
Login Screen: Allows registered users to access their accounts.
Home Screen: Provides subject selection tailored to Grade 12 topics.
Quiz Screen: Presents multiple-choice questions relevant to the chosen subjects.
Result Screen: Displays scores and detailed feedback on performance.
Future Enhancements
User Accounts: To offer personalized experiences based on individual learning paths.
Leaderboards: To encourage competitive engagement among users.
Database Integration: For dynamic content updates, ensuring access to the latest materials.
